System results

This website provides information about board games, including descriptions and brief gameplay instructions. It allows users to explore and learn on their own while waiting for a Game Master (GM) to explain the game. This addresses a pain point where the store‚Äôs staff is insufficient to assist all customers. The website includes the following features:

Home

  • Displays the top 5 most popular board games based on views.
  • Shows the total number of board games available in the database.

All Board Games

  • Displays all board games in the database.
  • Allows users to search for board games.
  • Enables sorting of games in ascending or descending order.
  • Allows filtering by game category.
  • Displays game details, descriptions, brief gameplay instructions, game expansions (if any), and tutorial videos.

Admin

  • Includes a login system.
  • Allows adding, deleting, and editing game information.
